Cluster-level monitoring:   
- Node functions
- Node availability
- Node resource usage
- Number of pods running

- Kubernetes metrics

    - Pod status, restarts
    - CPU, Memory usage
    - Network usage
- Container metrics

    - Network traffic/errors
- Application metrics
